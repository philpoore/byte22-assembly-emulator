Welcome to Byte22 Assembly Emulator.

A simple Assembly Emulator built with HTML5 and Angular.js

$ git clone git@github.com:philpoore/byte22-assembly-emulator.git a
$ cd a
$ open index.html

Examples in the "examples/" folder

Commands

log - Debug Command, log arguments
inc %r - Increment Register
dec %r - Decrement Register
add %ra %rb - Add %ra %rb Registers, result in %ra
sub %ra %rb - Subtract %ra %rb Registers, result in %ra
mul %ra %rb - Multiply %ra %rb Registers, result in %ra
div %ra %rb - Multiply %ra %rb Registers, result in %ra
load %r %c - Load %c contstant into %r
mov %ra %rb - Move Register %rb into Register %ra
mov %r [%m] - Move Memory %m into Register %r
mov [%m] %r - Move Register %r into Memory %m
mov [%ra] %rb - Move Register %rb into Memory at location Register %ra
jump label - Jump to labeled statement
cmp %r %c - Compair Register %r to Constan %c, result in %se
cmp %ra %rb - Compair Register %ra to Register %rb, result in %se
je label - Jump to label if %se is true
jn label - Jump to label if %se is false
